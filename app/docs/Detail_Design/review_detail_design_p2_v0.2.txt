[NEW-YEL-01] dedup.py の定数が config.py と二重定義になっている
該当箇所：§2-1 processor/dedup.py / §6 config.py

  §2-1 dedup.py
    DEDUP_THRESHOLD = 0.80
    DEDUP_WINDOW_DAYS = 7

  §6 config.py
    DEDUP_THRESHOLD   = 0.80
    DEDUP_WINDOW_DAYS = 7

  v0.2 で YEL-01（設定の一元化）対応として auth/users.py を config.py から
  インポートする方式に修正した。同じ方針が dedup.py には適用されておらず、
  設定値（閾値・比較期間）を変更したい場合は config.py と dedup.py の
  両方を修正する必要が生じる。

  修正方針：
    § 2-1 dedup.py の先頭から DEDUP_THRESHOLD / DEDUP_WINDOW_DAYS の定義を削除し、
    以下のインポートに置き換える。
      from config import DEDUP_THRESHOLD, DEDUP_WINDOW_DAYS
