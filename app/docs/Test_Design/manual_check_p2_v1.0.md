## 確認前の準備

1. バックエンドを起動（`uvicorn main:app --reload` 等、ローカル環境）
2. フロントエンドを起動（`npm run dev` 等）
3. バッチを最低1回実行済み（記事・試合日程データがDBにある状態）にしておく
