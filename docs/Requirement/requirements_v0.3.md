# 要件定義書

- **バージョン**: v0.3
- **ステータス**: Draft
- **作成日**: 2026-05-10
- **対象**: NBAニュース翻訳アプリ（個人利用）

---

## 1. プロジェクト概要

| 項目 | 内容 |
|---|---|
| 目的 | 海外NBAニュースを日本語で快適に読む |
| 利用者 | 個人（非公開） |
| 利用形態 | Webブラウザ（ローカルサーバー） |
| 更新頻度 | 1日1〜2回（朝・夕の定時バッチ） |
| お気に入りチーム | San Antonio Spurs（1チーム） |
| 将来拡張 | NFL対応（優先度低・フェーズ2） |

---

## 2. 機能要件

| ID | 機能名 | 内容 | 優先度 |
|---|---|---|---|
| F-01 | ニュース取得 | 公式RSS / 認可API経由でNBAニュースを定時取得 | 高 |
| F-02 | 日本語翻訳・要約 | 見出し＋本文をClaude APIで要約・日本語化（300〜500字程度） | 高 |
| F-03 | カテゴリ自動分類 | トレード / 契約 / 試合結果 / コラム の4分類をClaude APIで自動判定 | 高 |
| F-04 | お気に入りチームフィルタ | 設定した1チーム（Spurs）に関連するニュースのみ表示する切替機能 | 高 |
| F-05 | ネタバレ防止 | 試合結果カテゴリの詳細スコア・スタッツをデフォルト非表示（クリックで解除） | 高 |
| F-06 | 元記事リンク | 全文は元サイトへのリンクで参照（著作権対応） | 高 |
| F-07 | API使用上限管理 | Claude APIの月次コスト上限を設定、超過時は翻訳スキップ＋アラート表示 | 中 |
| F-08 | NFL拡張対応 | スポーツ種別を切り替えられる構造にしておく（フェーズ2実装） | 低 |

### 表示仕様（F-02補足）

- 見出し（翻訳済み）
- 要約本文（Claude APIが要約して日本語化、300〜500字程度）
- 元記事へのリンク
- 記事の全文翻訳・全文保存は行わない（著作権対応）

### ネタバレ防止仕様（F-05補足）

- 対象カテゴリ：「試合結果」
- 非表示対象：勝敗・スコア・ボックススコア・個人スタッツ
- デフォルト：非表示（ぼかし表示）
- 解除：「スコア・スタッツを表示」ボタンをクリックで解除
- 設定：アプリ全体でON/OFFを切替可能

---

## 3. 非機能要件

| 項目 | 内容 |
|---|---|
| 法的制約 | スクレイピング禁止。RSSまたは公式APIのみ使用。記事全文の翻訳・保存不可 |
| コスト制約 | Claude APIは月上限を設定。無料RSS優先、有料APIは最小化 |
| データ保持 | 取得済みニュースはローカルDB（SQLite等）に保存。最新30日分 |
| セキュリティ | APIキーは環境変数で管理。ローカルのみアクセス可 |
| パフォーマンス | ページ初期表示は1秒以内（キャッシュ済みデータ表示） |

## 5. 主要リスクサマリー（手順2で詳細調査予定）

### 法的リスク
- RSSフィードの利用規約確認必須（ESPN・NBA.com等）
- 翻訳は要約のみ（全文翻訳・保存は著作権侵害の恐れ）
- 個人利用であっても各サービスの利用規約に従う

### コストリスク
- Claude API：月次上限の設定必須
- 翻訳対象を要約に絞りトークン数を最小化
- 有料ニュースAPIは使用しない方針

### 技術リスク
- RSSフィード提供停止・仕様変更の可能性
- カテゴリ自動分類の精度（Claude APIのプロンプト設計で対応）
- ネタバレ判定の精度（スコア・スタッツの検出漏れ）
