## 1. プロジェクト概要

海外NBAニュースサイト（Hoops Rumors等）のRSSフィードを定時取得し、Claude APIで日本語要約・カテゴリ分類して表示するWebアプリ（ローカルサーバー）を開発する。

| 項目 | 内容 |
|---|---|
| 目的 | 海外NBAニュースを日本語で快適に読む |
| 利用者 | 個人（非公開） |
| 利用形態 | Webブラウザ（ローカルサーバー） |
| 参照ドキュメント | 要件定義書 v0.6（`requirements_v0.6.md`） |
