---

## 1. プロジェクト概要

海外NBAニュースサイト（ESPN等）のRSSフィードを定時取得し、Claude APIで日本語要約・カテゴリ分類して表示するWebアプリ（ローカルサーバー）を開発する。

| 項目 | 内容 |
|---|---|
| 目的 | 海外NBAニュースを日本語で快適に読む |
| 利用者 | 個人（非公開） |
| 利用形態 | Webブラウザ（ローカルサーバー） |
| お気に入りチーム | San Antonio Spurs |
| 参照ドキュメント | 要件定義書 v0.3 |

### 手順2：リスク事前調査 🔄

- **アウトプット**: リスク調査レポート（簡易版）
- **成果物**: `risk_report_v0.1.md`（作成予定）
- **目的**: 設計着手前に法的・金銭的リスクの致命的な問題がないことを確認する
- **調査項目**:

#### 法的調査
- [ ] ESPN RSS フィードの利用規約確認（個人利用の定義、商用・非商用の区別）
- [ ] NBA.com RSS フィードの利用規約確認
- [ ] その他利用予定RSSソースの利用規約確認
- [ ] 日本著作権法における「要約表示」の扱いの確認
- [ ] 翻訳要約の表示が引用の範囲内かどうかの確認

#### コスト調査
- [ ] Claude API（claude-sonnet-4系）の料金体系確認
- [ ] 1日2回取得 × 想定記事数（20〜30件/回）× 要約トークン数での月次コスト試算
- [ ] Claude API の月次上限設定方法の確認
- [ ] 無料RSSで賄えるニュースソースの範囲確認

#### 技術調査
- [ ] ESPN・NBA.com のRSSフィード提供状況の確認
- [ ] RSSフィードに含まれる情報量（本文量）の確認
- [ ] チームフィルタに必要なメタ情報（チーム名タグ等）がRSSに含まれるか確認

## 5. 制約・前提条件

| 区分 | 内容 |
|---|---|
| 法的制約 | スクレイピング禁止。RSSまたは公式APIのみ使用。記事全文の翻訳・保存不可 |
| コスト制約 | Claude APIは月次上限を設定。無料RSS優先、有料APIは最小化 |
| 技術制約 | ローカル環境での動作のみ。外部公開不可 |
| 運用制約 | 個人利用・非公開。第三者への提供・転用禁止 |
